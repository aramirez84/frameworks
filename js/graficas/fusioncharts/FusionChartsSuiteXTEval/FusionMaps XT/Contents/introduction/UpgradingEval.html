Upgrading from FusionMaps XT Evaluation

This section is meant for users who have implemented the Evaluation version of FusionMaps XT in their application, and are now upgrading to the commercial license.

There is no feature limitation in the Evaluation version of FusionMaps XT, except for the fact that the evaluation version maps are watermarked. If you have been using the evalution of FusionMaps XT in your application and are now upgrading to a commercial license of FusionMaps XT, you'll need to do the following:

  1. Replace Evaluation version JavaScript files with newly purchased JavaScript files (present in Download Package > Maps folder)
  2. If you are using a version earlier to FusionMaps v3.3.0, make sure you have copied FusionCharts.HC.js and FusionCharts.HC.Maps.js, the map definition js files like FusionCharts.HC.world.js and jquery.min.js (along with FusionCharts.js) from the Licensed Download Package > Maps folder
  3. Replace Evaluation version Flash (SWF) files with newly purchased Flash (SWF) files (present in Download Package > Maps folder)
  4. Clear your browser cache - very important step.

When you now view your application, you will not find any stamps on your maps.

In case, you are also using charts from FusionCharts XT v3.2.2 SR-5 or earlier version, you need to download or update to the latest FusionCharts XT pack and copy FusionCharts.HC.Charts.js from the Charts folder of the downloaded FusionCharts XT pack.

Moreover, if you are using charts from PowerCharts XT v3.2.1 SR-1 or earlier version, you need to download or update to the latest PowerCharts XT pack and copy FusionCharts.HC.PowerCharts.js from the Charts folder of the downloaded PowerCharts XT pack.

Moreover, if you are using charts from FusionWidgets XT v3.2 SR-2 or earlier version, you need to download or update to the latest FusionWidgets XT pack and copy FusionCharts.HC.Widgets.js from the Charts folder of the downloaded FusionWidgets XT pack.
