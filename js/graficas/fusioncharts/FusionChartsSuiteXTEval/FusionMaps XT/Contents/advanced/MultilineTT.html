Putting tooltip in multiple lines
 
FusionMaps XT allows you to put your tooltips in multiple lines with a little manual tweaking. Consider the XML below:
 
  • XML
  • JSON
<map>
 <data>
  <entity id="NA" value="515" tooltext="North America{br}Population: 515{br}Position:4"/>
  <entity id="SA" value="373" tooltext="South America{br}Population: 373{br}Position:5"/>
  <entity id="AS" value="3875" tooltext="Asia{br}Population: 3875{br}Position:1"/>
  <entity id="EU" value="727" tooltext="Europe{br}Population: 727{br}Position:3"/>
  <entity id="AF" value="885" tooltext="Africa{br}Population: 885{br}Position:2"/>
  <entity id="AU" value="32" tooltext="Australia{br}Population: 32{br}Position:6"/>
 </data>
</map>
{
  "map": {},
  "data": [
    {
      "id": "NA",
      "value": "515",
      "tooltext": "North America{br}Population:515{br}Position:4"
    },
    {
      "id": "SA",
      "value": "373",
      "tooltext": "South America{br}Population:373{br}Position:5"
    },
    {
      "id": "AS",
      "value": "3875",
      "tooltext": "Asia{br}Population:3875{br}Position:1"
    }
	{
      "id": "EU",
      "value": "727",
      "tooltext": "Europe{br}Population:727{br}Position:3"
    }
	{
      "id": "AF",
      "value": "885",
      "tooltext": "Africa{br}Population:885{br}Position:2"
    }
	{
      "id": "AU",
      "value": "32",
      "tooltext": "Australia{br}Population:515{br}Position:4"
    }
  ]
}

In the above XML/JSON, we're first setting 3 lines tooltip content for each <entity>. To separate each line we use the {br} pseudo code, which is equivalent to <BR> HTML tag in FusionMaps XT for tooltips.

Applying HTML to labels

Similarly, you can apply line breaks to your data labels also. Consider the XML/JSON below:

  • JSON

 <data>
  <entity id="NA" value="515" displayValue="North America{br}Population: 515{br}Position:4"/>
